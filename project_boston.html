Overview

This project involved designing and deploying an end-to-end machine learning pipeline to predict housing prices using the Boston Housing Dataset. The objective was to create a web-based tool that could help users estimate property values based on key input features.

Problem Statement

Accurately predicting property prices is critical for buyers, sellers, and real estate investors. The goal was to build a regression-based model that takes housing features as input and returns a reliable price estimate.

Approach & Methodology

  • Data Exploration & Cleaning:
    • Performed in-depth EDA to understand relationships between features such as crime rate, number of rooms, and distance to employment centers.
    • Handled missing values and outliers to ensure data integrity and consistency.
  • Feature Engineering & Selection:
    • Applied correlation analysis and variance thresholding to select relevant features.
    • Normalized continuous variables to improve model performance.
  • Model Development:
    • Implemented Linear Regression as the base model using Scikit-learn.
    • Conducted hyperparameter tuning via cross-validation to improve prediction accuracy.
  • Evaluation:
    • Achieved a solid performance with metrics such as RMSE and R¬≤ on both training and test datasets.
    • Visualized residuals to validate model assumptions and detect potential bias.
  • Deployment:
    • Developed a lightweight web application using Flask.
    • Deployed the model on Heroku, enabling real-time user interaction via a simple front-end where users input data and receive a predicted price instantly.

Key Learnings & Highlights

  • Gained practical experience in building an ML pipeline from scratch ‚Äî from data preprocessing to cloud deployment.
  • Strengthened understanding of linear regression assumptions and model diagnostics.
  • Built a working application that bridges data science with real-world usability.
